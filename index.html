Introdução

HTML é o código que você utiliza para estruturar uma página web e seu conteúdo. Ele não é uma linguagem de programação, mas sim uma linguagem de marcação. Basicamente, o HTML define a estrutura do conteúdo da página, como parágrafos, imagens, tabelas e links.

Tags < h1 > até < h6 >

As tags h1 até h6 são tags de cabeçalho, que representam seis níveis de importância, do maior nível para o menor, sendo h1 usada normalmente para o nivel mais alto de título, como o título principal desse projeto por exemplo. Já h2 esta sendo usado como a introdução, ou poderia ser usado como um subtítulo, h3 como subseções e assim por diante.
Exemplo:

nível 1

nível 2

nível 3

nível 4

nível 5
nível 6

Tag < p >

A tag < p > é usada para definir um parágrafo em HTML. Ela cria um bloco de conteúdo que quebra automaticamente as linhas de acordo com o espaço disponível. Em mídias visuais, parágrafos são representados como blocos indentados de texto, separados por linhas em branco. Essa tag é essencial para estruturar o texto em um documento HTML.

 índice

Tag < mark >

A tag < mark > é usada para destacar partes do texto em HTML. Ela representa um trecho de destaque, como uma sequência de texto relevante em um contexto específico. Por exemplo, você pode usá-la em uma página de resultados de busca, onde todas as instâncias da palavra pesquisada recebem destaque, em outras palavras, ela funciona como a conhecida caneta "grifa texto."

 índice

Tag < small >

A tag < small > em HTML é usada para reduzir o tamanho do texto. Ela define um trecho de texto menor, geralmente usado para indicar informações secundárias, como direitos autorais ou comentários laterais. Outro exemplo de onde ela esta sendo usada nesse projeto, são nos links de retorno "índice", como logo após esse parágrafo.

Tag < i >

A tag < i > em HTML representa uma parte do texto que é destacada do restante por algum motivo. Geralmente, ela é usada para termos técnicos, expressões de outros idiomas ou pensamentos. O padrão é que o conteúdo dentro dessa tag seja apresentado em itálico. Como exemplo o pensamento de Gorndon Moore. “Se a indústria automobilística avançasse tão rapidamente quanto a indústria de semicondutores, um Rolls Royce faria meio milhão de milhas por galão, e seria mais barato jogá-lo fora do que estacioná-lo.”

Tag < u >

A tag < u > é uma das tags de formatação de texto disponíveis no HTML. Ela é usada para sublinhar o texto dentro de um elemento específico. Quando aplicada, a tag < u > adiciona uma linha horizontal abaixo do texto, destacando-o visualmente.

 índice

Tag < strong >

A tag < strong > em HTML dá forte importância ao texto e é tipicamente mostrada em negrito. Ela é usada para destacar conteúdo de grande relevância, como alertas ou informações urgentes.

Tag < ul >

tag < ul > em HTML representa uma lista de itens sem ordem rígida. Isso significa que é uma coleção de itens que não segue uma ordenação numérica específica, e a posição dos itens na lista é irrelevante. Caracteristicamente, os itens em uma lista desordenada são exibidos com um marcador, que pode ter várias formas, como um ponto, um círculo ou um quadrado.Exemplo:

  • Ivan
  • Tatiane
  • Victor

 Índice

Tag < ol >

A tag < ol > em HTML é usada para criar uma lista ordenada, ou seja, com elementos que precisam aparecer em sequência. Exemplo:

  1. Ivan
  2. Tatiane
  3. Victor

 Índice

Tag < li >

A tag < li > em HTML é usada dentro de uma lista < ul > (lista desordenada) ou < ol > (lista ordenada), vistas acima, para definir cada item da lista. Essa tag ajuda a organizar o conteúdo e torna as informações mais fáceis de entender para os usuários. Exemplo:
< ol >
< li > item da lista < /li >
< li > item da lista < /li >
< li > item da lista < /li >
< /ol >

Índice

Tag < Sub >

A tag < sub > em HTML é usada para adicionar texto subscrito ao documento. Quando você usa a tag < sub >, o texto aparece meio caractere abaixo da linha normal e, às vezes, é renderizado em uma fonte menor. Essa tag é comumente usada para fórmulas químicas,por exemplo, como H2O, ou para marcar números de notas de rodapé. Também pode representar variáveis em fórmulas matemáticas, como xn.

Tag < blockquote >

O Elemento HTML < blockquote > (ou Elemento HTML de citação de bloco) indica que o texto incluído é uma longa citação. Normalmente, este é processado visualmente pelo recuo. A URL para a fonte da citação pode ser dada usando o atributo cite, enquanto uma representação de texto da fonte pode ser dada usando o < cite > elemento.

Tag < a >

A tag < a > é utilizada para criar links ou âncoras para outras páginas ou recursos. Ela permite vincular uma página a outra, utilizando o atributo "href". Exemplo:
